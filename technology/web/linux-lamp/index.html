LAMP 编译安装三部曲 ​

LAMP 是一种常用于搭建网站和应用程序的服务器架构，它由 Linux、Apache、MySQL（或MariaDB）、PHP 四个开源软件组成，也分别代表了操作系统、Web 服务器、数据库以及服务器端脚本语言。 与 LNMP 架构类似，区别在于使用的 Web 服务器组件。

本教程将用三步带你完成 LAMP 架构的编译安装。

  1. 第一步 编译安装 httpd 服务器
  2. 第二步 编译安装mariadb数据库环境
  3. 第三步 编译安装PHP
